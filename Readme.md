# Glow

A Physically Based 3D Model Renderer.

## Demo

![screenshot](Screenshot.png)

## Build

On Windows with vcpkg installed

```sh
mkdir build
cd build
cmake ..
# run demo scene with:
./glow.exe
```
I have not tested on other platforms
